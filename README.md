# Dia Animation Demo

Demo for the Dia browser background animtion.

[Demo Preview](https://dia-animation.vercel.app/)

Used：
- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide

## Commands

```bash

pnpm install

pnpm dev

```
